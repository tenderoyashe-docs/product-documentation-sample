# User Guide: TaskFlow Project Dashboard

## Overview
TaskFlow is a project management platform designed to help teams track tasks, manage deadlines, and monitor progress in real time.

The Project Dashboard provides a centralized view of active tasks, project status, and team workload.

---

## Accessing the Dashboard
1. Log in to TaskFlow.
2. From the main navigation menu, click **Dashboard**.
3. Select the desired project from the dropdown list.

---

## Dashboard Components

### 1. Task Summary Panel
Displays:
- Total tasks
- Completed tasks
- Overdue tasks
- Tasks due today

### 2. Progress Chart
A visual representation of project completion percentage.

### 3. Team Workload Section
Shows tasks assigned per team member to identify capacity issues.

---

## Filtering Tasks
Users can filter tasks by:
- Status
- Priority
- Due date
- Assigned team member

---

## Exporting Reports
1. Click **Export** in the top-right corner.
2. Select format (PDF or CSV).
3. Confirm export.

The report will download automatically.
---

## Permissions

The Project Dashboard is accessible to:

- Project Managers
- Team Members assigned to the project
- Administrators

Users without project access will not see the dashboard in the navigation menu.
